Mais Sobre o Flask

Flask é um micro framework web escrito em Python. É classificado como microframework porque não requer ferramentas ou bibliotecas específicas. Não possui camada de abstração de banco de dados, validação de formulário ou quaisquer outros componentes onde bibliotecas pré-existentes de terceiros forneçam funções comuns. Serve para desenvolver Web Apps, APIs, WebSites, etc.
